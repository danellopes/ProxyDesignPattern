### Example of the Proxy Design Pattern

This example of the Proxy Design Pattern pattern was develop using C# language and Salesforce Apex language.

The Proxy Design Pattern provides a surrogate or placeholder for another object to control access to it. This pattern is particularly useful in scenarios such as lazy initialization, access control, logging, or when working with remote objects.

In this repository, you'll find examples demonstrating how the Proxy Design Pattern can be implemented in both C# and Salesforce Apex, showcasing its versatility across different programming environments.
